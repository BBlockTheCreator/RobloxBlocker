How to Install RobloxBlocker

Follow these steps to manually install the RobloxBlocker extension on Microsoft Edge or Google Chrome:

  1. Download the ZIP file above.
  2. Extract the contents to a folder on your computer.
  3. Open your browser and go to the Extensions page:
    • Chrome: chrome://extensions
    • Edge: edge://extensions
  4. Enable “Developer mode” (top-right corner).
  5. Click “Load unpacked”.
  6. Select the folder where you extracted the ZIP.
  7. Done! The extension is now active.
